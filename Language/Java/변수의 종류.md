## 변수의 저장 위치
![[다운로드.png]]
- JVM 내부에서 Runtime Data Area에 위치한다.
	- Runtime Data Area: 프로그램이 실행되는 동안 필요한 데이터를 보관하고 관리하는 영역
- 힙 영역: new 로 생성된 모든 객체가 저장되는 공간
- 스택 영역: 메서드 호출마다 생성되는 임시 공간, 지역변수와 메서드 정보가 저장된다
- 메서드 영역: 클래스 로더가 로딩한 클래스 정보 및 정적 변수를 저장한다.

JVM 구조의 나머지 부분(PC 레지스터, 네이티브 메서드 스택)이나 GC의 세부 작동 원리, 클래스 로더 등은 추후 학습 예정
